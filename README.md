Developer Technical Test - Coffee Machine over the cloud
## Overview
This repository contains the implementation for an imagiary internect-connected coffee machine.
The project is build using ASP.Net Core 10.0

This specific branch utilizes OpenWeatherMap's API 

### Prerequisites
.NET SDK
An IDE (I used Visual Studio)

### Configuration
1. Open 'appsettings.json'
2. Replace "YOUR_OPENWEATHERMAP_API_KEY" with a valid key which you can get from https://openweathermap.org/api
